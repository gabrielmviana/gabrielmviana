## 🟦 Projeto ATTALUS — Homelab Corporativo

Laboratório prático simulando a infraestrutura de uma empresa, com foco em redes, servidores e organização corporativa.

### 🔧 Ambiente

| Componente | Detalhes |
|---|---|
| Roteador | Mikrotik RB750Gr3 |
| Host principal | Notebook |
| Virtualização | VirtualBox |
| VMs | Windows Server · Linux Server |

### ✅ Implementações realizadas

- Segmentação de rede com VLANs
- Configuração de NAT
- Regras iniciais de firewall
- Isolamento de ambiente de laboratório

### 🎯 Objetivos

- Simular ambientes corporativos reais
- Aplicar conceitos de redes e infraestrutura
- Documentar implementações técnicas
- Desenvolver raciocínio de troubleshooting
